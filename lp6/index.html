About learning plan 6


Overview

The overview of lesson 6 was learning how to do 2d and 3d transformations using practices given to us.


Goals of the assignment

The goal of this assignment was to use both 2d and 3d transformations. I learned how to use and implement 2d and 3d transitions. I learned some new styles and properties such as the rotate methods, translate and skew. The practices had us use those properties in the projects. For the 3d part, I learned how to use methods such as translate, scale, and rotate again.


Reflection

Learning plan 6 was kind of difficult to do and figure out in the beginning but I understood it more as I kept doing it and using sources. Both 2d and 3d were difficult to learn I think because of the properties.
